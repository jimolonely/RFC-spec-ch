1999年

**摘要**

-----

Hypertext Transfer Protocol(HTTP) 是应用层的协议，用在分布式、协作、超媒体信息系统。
是一个通用的、无状态的协议，可以用在很多任务，不只是超文本，比如名称服务，分布式对象管理系统，
只需要扩展其request方法，错误码和header。HTTP的特点是允许系统和数据传输相互独立，因为HTTP不
影响数据本身，只是数据的类型和协调。

从1990年开始，HTTP已经被用于万维网。
